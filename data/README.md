Dataset Setup Instructions

This project uses a CSV-indexed fashion image dataset.

Required files (NOT included in this repository):
- images.csv
- images_compressed.zip

Steps to run:
1. Download the dataset from Kaggle:
   Fashion Product Images Dataset (clothing-dataset-full)
2. Place both files in the root directory:
   - images.csv
   - images_compressed.zip
3. Run the notebook:
   notebook/train.ipynb

The notebook will automatically extract images into the /images folder.
